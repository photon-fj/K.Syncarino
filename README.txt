バックアップソフトのフリー版をテストリリースします。
皆さんどうぞ使ってみてください。

要望事項等ありましたら github にあげていただけると幸いです。


・このソフトの特徴
 2つのディレクト間でファイル同期をとるバックアップソフトです。
 大切なデータを長期間保管が目的です。


 SHA-256 ハッシュ値を保持し、壊れていないかを検証可能
 ファイルの内容が正しいかどうかを、日付やサイズ以外にファイルのハッシュ値でも判定します。

 2つのディレクトリに“同じファイル”を保持し、壊れていれば修復ができます
 2つのディレクトリ間のファイルに差異があった場合、ハッシュに基づき正しいファイルを判定し、復旧操作ができます。

 複数のディレクトリペアを登録・管理可能
 1組だけでなく、用途ごとに複数のペアを追加して運用できます。


・動作環境
	Windows11
	.NET9.0 が必要です。未インストールの場合、本ソフトインストール時にダウンロードされます。
