皆さんどうぞ使ってみてください。

要望事項等ありましたら github にあげていただけると幸いです。


・このソフトの特徴
	 2つのディレクト間でファイルを同期するバックアップソフトです。
	 特に、長期間にわたる大切なデータの破壊検知と保全を目的としています。

	 ファイルの内容の正しさ（破壊されていないか）を、以下の複数の情報で判定します。
	  ・ファイル作成日
	  ・ファイル更新日
	  ・ファイルサイズ
	  ・ファイルハッシュ値(SHA-256)

	 これら情報をデーベースに保管し、初回以降はデータベースと照合することでファイルの正しさを判定します。
	 差異があったファイルは画面上から修復操作(指示に従い正しいファイルで上書き)ができます。

	 複数のディレクトリペアを登録・管理可能


・動作環境
	 Windows11
	 .NET9.0 が必要です。
